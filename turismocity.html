Project Overview

Turismocity is a travel booking platform that allows users to compare flights, hotels, and travel deals.

The goal of this project was to redesign the desktop experience to improve usability, simplify navigation, and reduce friction in the booking process. The original platform presented several issues such as confusing navigation, overwhelming search results, and lack of pricing transparency.

User Research & Problem Discovery

To understand the main usability issues, I conducted a UX audit that included usability testing, analysis of user feedback, and a review of competitor travel platforms.

This research revealed several key problems that negatively impacted the user experience.

Key Usability Issues

  • •Confusing navigation structure
  • •Overwhelming search results with poor filtering options
  • •Lack of transparent pricing information
  • •Complex and fragmented booking process

Before

  • Poor navigation structure
  • Cluttered and overwhelming search results
  • Hidden or inconsistent pricing
  • Complicated booking flow
  • Low user trust and high drop-off rates

After

  • Clear and organized navigation
  • Improved search results with smart filters
  • Transparent pricing across the booking process
  • Streamlined and intuitive booking journey
  • More trustworthy and focused user experience

Key Design Solutions

  • Simplified navigation with clear categories
  • Improved search functionality with advanced filters
  • Transparent pricing throughout the booking flow
  • A simplified, linear booking journey
  • Trust-building elements such as reviews and ratings
  • Personalized travel suggestions
  • Clear cancellation and refund policies

Design Approach

I created wireframes and high-fidelity mockups focused on improving usability and clarity.

The information architecture was reorganized to better match user expectations and mental models. Design iterations were guided by usability insights to reduce cognitive load and make it easier for users to compare travel options and complete bookings.

Final Outcome

The redesigned desktop experience improves navigation, transparency, and booking efficiency. It offers users a smoother planning journey, from discovery to confirmation, aligned with modern UX best practices.

Learnings

This project highlighted the importance of simplifying complex booking processes and presenting information in a clear and transparent way. It also reinforced how better filtering, navigation, and trust signals can significantly improve the overall user experience.
